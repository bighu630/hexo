## 1. 五大基本情绪模型

### 1.1 Ekman的六种基本情绪

**Paul Ekman** 的基本情绪理论是情绪心理学领域最具影响力的理论之一。基本情绪被认为具有以下特征：
- 普遍性的面部表情
- 跨文化一致性
- 快速的自动加工过程
- 先天预接线（pre-wired）的神经机制

**六种基本情绪及其特征：**

| 情绪 | 效价 | 唤醒度 | 核心面部特征 | 核心生理反应 |
|------|------|--------|-------------|-------------|
| **喜悦 (Joy)** | 正向 | 低-中 | 微笑（嘴角上扬，眼轮匝肌收缩） | 心率降低或稳定，皮肤电导降低 |
| **愤怒 (Anger)** | 负向 | 高 | 眉间降低，眉心收缩，嘴唇紧闭/张开 | 心率升高，血压上升，皮肤电导升高 |
| **悲伤 (Sadness)** | 负向 | 中 | 眉头上扬，嘴角下垂，眼眶湿润 | 心率降低，呼吸减慢 |
| **恐惧 (Fear)** | 负向 | 高 | 眼睛睁大，眉毛上扬，嘴唇微张 | 心率急剧升高，皮肤电导显著升高，颤抖 |
| **厌恶 (Disgust)** | 负向 | 中 | 皱鼻，上唇上扬，面部整体下压 | 心率降低或稳定，呼吸变浅 |
| **惊讶 (Surprise)** | 中性-正向 | 高 | 眉毛弯曲，眼睛睁大，下颌张开 | 心率升高，短暂停顿后的呼吸增强 |

**扩展：** Ekman后期将"兴奋/唤起（Arousal）"作为一个独立维度添加，并在与Wallace Friesen的合作中发现基本情绪在面部肌肉层面具有独特的运动单元（Action Units, AU）组合。

### 1.2 基本情绪的扩展

**八种基本情绪（扩展版）：** 部分研究者（包括Ekman本人后续研究）将以下两种情绪纳入基本情绪范畴：

- **轻蔑 (Contempt)**：最早由Ekman和Friesen在1986年确认，具有独特的面部标志（左嘴角上扬），在跨文化研究中与愤怒、厌恶、悲伤等可区分

- **娱乐/趣味 (Amusement)**：在喜悦的基础上增加了更强的唤醒度和认知评价成分

## 2. Plutchik情绪轮模型

**Robert Plutchik**（1980, 2001）提出了著名的"情绪轮"（Emotion Wheel），定义了8种原始情绪，并将它们排列为4对对立的极性。

### 2.1 八种原始情绪及其对立对

```
        喜悦 (Joy) ←————————→ 悲伤 (Sadness)
            ↑                    ↓
            ↑                    ↓
    信任 (Trust) ←———→ 厌恶 (Disgust)
            ↑                    ↓
            ↑                    ↓
    恐惧 (Fear) ←——————→ 愤怒 (Anger)
            ↑
            ↑
        惊讶 (Surprise)
                              （与期待有关，居于中心）
```

### 2.2 Plutchik的层级强度模型

Plutchik提出每种原始情绪存在从轻微到强烈的**强度梯度**：

| 原始情绪 | 强度1（轻微） | 强度2（中等） | 强度3（强烈） |
|---------|-------------|-------------|-------------|
| 喜悦 | 舒适 (Serenity) | 喜悦 (Joy) | 狂喜 (Ecstasy) |
| 悲伤 | 忧郁 (Pensiveness) | 悲伤 (Sadness) | 悲痛 (Grief) |
| 恐惧 | 安静 (Apprehension) | 恐惧 (Fear) | 恐怖 (Terror) |
| 愤怒 | 烦恼 (Annoyance) | 愤怒 (Anger) | 狂怒 (Rage) |
| 厌恶 | 厌倦 (Boredom) | 厌恶 (Disgust) | 憎恶 (Loathing) |
| 惊讶 | 分心 (Distraction) | 惊讶 (Surprise) | 震惊 (Amazement) |
| 信任 | 接受 (Acceptance) | 信任 (Trust) | 崇拜 (Admiration) |
| 期待 | 兴趣 (Interest) | 期待 (Anticipation) | 警觉 (Vigilance) |

### 2.3 情绪混合（Emotional Mixtures）

Plutchik理论的一个重要贡献是提出了**情绪混合**（类似色彩混合）的概念：
- **喜悦 + 恐惧 = 愧疚/有罪感（Guilt）？**（有争议）
- **喜悦 + 期待 = 乐观/希望（Optimism/Hope）**
- **愤怒 + 厌恶 = 轻蔑（Contempt）**
- **恐惧 + 惊讶 = 敬畏（Awe）**
- **悲伤 + 厌恶 = 悲观（Pessimism）？**
- **愤怒 + 恐惧 = 攻击性（Aggression）？**

## 3. 情绪维度模型

### 3.1 Russell的环状模型（Circumplex Model）

**James A. Russell**（1980, 2003）在其环状情绪模型中提出，所有情绪可以定位在一个由两个正交维度构成的圆周上：

**维度1：效价（Valence）**
- 从负向（Negative）到正向（Positive）
- 反映情绪的主观愉悦度

**维度2：唤醒度（Arousal）**
- 从低唤醒度（Low Arousal）到高唤醒度（High Arousal）
- 反映神经系统的激活程度

**环状模型中的主要情绪位置：**

```
              高唤醒度
                 ↑
        兴奋 ←——┼——→ 紧张
        (Excited)   (Tense)
           ↑    ↑    ↑
           ↑    ↑    ↑
    快乐 ←——┼——┼——→ 愤怒
    (Happy)  愉悦  (Angry)
    (Pleased) ↑    ↑
           ↑    ↓    ↑
           ↑    ↓    ↑
    满足 ←——┼——┼——→ 悲伤
    (Content) 低唤醒度  (Sad)
           ↓    ↓    ↓
        放松 ←——┼——→ 抑郁
        (Relaxed)  (Depressed)
                 ↓
              厌倦
              (Bored)
```

### 3.2 PAD情绪模型

**Albert Mehrabian**（1996）提出的PAD三维模型：

- **P - Pleasure（愉悦度）**：正向vs负向情感体验
- **A - Arousal（唤醒度）**：激活vs未激活状态
- **D - Dominance（支配度）**：控制vs被控制感

PAD模型可以更精细地区分一些在Russell模型中位置接近的情绪（如愤怒vs恐惧）。

### 3.3 情绪维度对照表

| 情绪 | Russell效价 | Russell唤醒度 | PAD愉悦度 | PAD唤醒度 | PAD支配度 |
|------|----------|-----------|---------|---------|---------|
| 喜悦 | + | + | +高 | 中 | 中-高 |
| 愤怒 | - | ++ | -中 | ++高 | +高 |
| 悲伤 | - | - | -高 | -低 | -低 |
| 恐惧 | - | ++ | -中 | ++高 | -低 |
| 厌恶 | - | 中 | -高 | 中 | 中 |
| 惊讶 | 中 | ++ | +中 | ++高 | -低 |
| 羞耻 | - | + | -高 | +中 | -低 |
| 焦虑 | - | ++ | -高 | ++高 | -低 |
| 自豪 | + | + | +高 | +中 | +高 |

## 4. 复合情绪的构成

复合情绪是由基本情绪组合而成的更复杂的情感体验。以下是各主要复合情绪的构成分析。

### 4.1 害羞/社会性焦虑（Shyness / Social Anxiety）

**心理学定义：** 害羞是当个体面对社会评价情境时产生的自我意识情绪，伴随着对负面评价的恐惧和社交回避倾向。

**成分构成：**
- **恐惧（Fear）**：40-50%——对负面社会评价的恐惧，对被拒绝的担忧
- **厌恶（Disgust）**：15-20%——通常指向自我（self-directed disgust），对自己的社会表现感到厌恶
- **喜悦（Joy）**：5-15%——对社交成功的期待（正向成分）；或在轻度害羞中，对情境幽默性的感知

**文献依据：** Crozier & Page (2012)；Henderson, Zimbardo & Carducci (2010)的研究表明，害羞的核心是"社交恐惧"加上"对自我暴露的羞耻感"，后者在轻度时表现为尴尬，严重时泛化为对社交场合的全面厌恶。

**Russell维度：** 负向效价（-），中低唤醒度（+）

### 4.2 内疚（Guilt）

**心理学定义：** 内疚是个体对自身行为的道德违反感到的痛苦情感，关注的是"我做了什么"，而不是"我是什么样的人"。

**成分构成：**
- **悲伤（Sadness）**：30-40%——对伤害他人/违反自我标准的痛苦
- **恐惧（Fear）**：20-30%——对被发现、被惩罚或关系破裂的预期恐惧
- **愤怒（Anger）**：10-20%——通常指向自我（self-anger），或对造成伤害的情境的愤怒
- **羞耻（Shame）**：0-30%——与羞耻高度相关但不等同（见下）

**文献依据：** Tangney (1991, 1996)的内疚羞耻区分研究（Guilt Shame and Proneness Scale）；Haidt (2003)的道德情绪理论；Keltner & Buswell (1996)的研究表明内疚包含"对自身行为的痛苦"和"修复关系的动机"两个核心成分。

**Russell维度：** 负向效价（-），低唤醒度（-），涉及自我指向的痛苦

### 4.3 嫉妒（Jealousy）

**心理学定义：** 嫉妒是对实际或想象威胁的占有性情绪反应，通常涉及三角关系（自我、伴侣、竞争对手）。

**成分构成：**
- **愤怒（Anger）**：30-40%——对竞争对手或背叛伴侣的敌意
- **恐惧（Fear）**：25-35%——对失去关系、情感或地位的恐惧
- **悲伤（Sadness）**：20-30%——对关系可能破裂的失落感
- **厌恶（Disgust）**：5-10%——对竞争对手或背叛行为的蔑视

**文献依据：** Parrott & Smith (1993)的嫉妒三维成分研究；Salovey (1991)的"嫉妒四成分"模型（敌意、焦虑、悲伤、恐惧）；Buss (2000)的进化心理学视角强调嫉妒中的愤怒和恐惧的进化功能。

### 4.4 羡慕/嫉妒（Envy）

**心理学定义：** 羡慕是对他人拥有自己渴望但缺乏的东西的情感，包含"向上社会比较"带来的痛苦。

**成分构成：**
- **悲伤（Sadness）**：35-40%——对自身缺乏某物的失落感
- **愤怒（Anger）**：25-30%——对他人拥有自己渴望之物的怨恨
- **恐惧（Fear）**：10-15%—���对差距进一步扩大的担忧
- **厌恶（Disgust）**：5-10%——通常指向自我（指向自身不足）或他人
- **欲望（Desire）**：20-25%——对渴望之物的想要（不属于Ekman六种，但在情绪研究中广泛使用）

**Envy vs. Jealousy 区分：**
- **Envy（羡慕）**：我想要你没有的东西（A想要B的成就）——二人关系
- **Jealousy（嫉妒）**：我害怕失去我已有的东西给你（A害怕C夺走B）——三角关系

**文献依据：** Parrott & Smith (1993)的羡慕嫉妒对比研究；Smith & Kim (2007)的羡慕综述研究。

### 4.5 羞耻（Shame）

**心理学定义：** 羞耻是当个体的整体自我被评价为不合格、失败或有所欠缺时产生的痛苦情绪，是"自我意识情绪"（self-conscious emotion）的核心。

**成分构成：**
- **厌恶（Disgust）**：30-40%——强烈的自我厌恶（self-disgust），通常指向整体自我
- **悲伤（Sadness）**：25-35%——深层的痛苦和自我价值的降低
- **恐惧（Fear）**：15-25%——对暴露、被贬低、被拒绝的恐惧
- **愤怒（Anger）**：10-20%——自我指向的愤怒，伴随无力感

**羞耻 vs. 内疚的区分：**
- **羞耻（Shame）**："我很糟糕"（全局性自我否定）→ 回避倾向更强
- **内疚（Guilt）**："我做了糟糕的事"（行为特定）→ 修复倾向更强

**文献依据：** Tangney & Dearing (2002)《Shame and Guilt》；Lewis (1971)的羞耻-内疚区分；Scheff (2000)的社会学视角。

### 4.6 自豪（Pride）

**心理学定义：** 自豪是当个体或个体的行为/成就被评价为有价值时产生的正向情感，是"自我意识情绪"的正向极。

**成分构成：**
- **喜悦（Joy）**：50-60%——核心的正向效价基底
- **愤怒（Anger）**（代理自豪）：10-20%——对蔑视的反感/自我防御（代理自豪/驕傲感）
- **惊讶（Surprise）**：5-10%——当成就超出预期时（特定情境）
- **支配度（Dominance）**：高——在PAD模型中，自豪是支配度最高的正向情绪

**类型区分：**
- **真实自豪（Authentic Pride）**：基于真实成就→伴随谦虚特质
- **代理自豪（Hubristic Pride）**：基于自我整体→伴随傲慢特质

**文献依据：** Tracy & Robins (2007)的自豪类型学；Weidman et al. (2016)的神经影像学研究。

### 4.7 轻蔑（Contempt）

**心理学定义：** 轻蔑是对感知到的他人劣等性的优越感，混合了厌恶和愤怒，是道德情绪的关键组成部分。

**成分构成：**
- **厌恶（Disgust）**：45-55%——对他人特质/行为的厌恶反应
- **愤怒（Anger）**：35-45%——对被蔑视对象的不满和优越感
- **喜悦（Joy）**：5-10%——对自身优越性的微小满足感

**文献依据：** Rozin, Lowery & Ebert (1994)的"道德厌恶"研究；Hutcherson & Gross (2011)的道德情绪研究；Ekman & Friesen (1986)的轻蔑面部表情研究。

### 4.8 焦虑（Anxiety）

**心理学定义：** 焦虑是对未来不确定威胁的预期性情绪，包含恐惧成分但持续时间更长、更弥散。

**成分构成：**
- **恐惧（Fear）**：40-50%——核心成分，预期性恐惧（apprehensive fear）
- **悲伤（Sadness）**：15-25%——对未来消极结果的预期性悲观
- **惊讶（Surprise）**：5-15%——对威胁突然出现或变化的不确定反应
- **厌恶（Disgust）**：5-10%——通常在特定焦虑类型中出现（如强迫症中的污染厌恶）

**亚型映射：**
- **社交焦虑** → 害羞成分构成
- **状态焦虑** → 高唤醒+恐惧+悲伤
- **特质焦虑** → 持续的恐惧基线+低愉悦度

**文献依据：** Barlow (2000, 2002)的"通用焦虑障碍"理论；Watson & Clark (1984)的负向情感（NA）研究；Barlow的"情绪人格"（emotional personality）理论。

### 4.9 孤独（Loneliness）

**心理学定义：** 孤独是对渴望的社会联结与实际社会联结之间的感知差距的主观痛苦体验（不等同于独处）。

**成分构成：**
- **悲伤（Sadness）**：35-45%——核心成分，核心是对联结缺失的痛苦
- **恐惧（Fear）**：15-25%——对社会进一步拒绝的担忧，对亲密的恐惧
- **愤怒（Anger）**：10-20%——对自身处境或导致孤独的他人的怨恨
- **厌恶（Disgust）**：5-10%——通常指向自我（self-alienation）或对孤独状态的厌恶
- **喜悦（Joy）**：5-10%——当偶尔连接发生时（短暂的正向体验）

**文献依据：** Cacioppo & Cacioppo (2018)的孤独神经生物学研究；Perlman & Peplau (1981)的社会孤独理论；Heinrich & Gullone (2006)的孤独与人格研究。

### 4.10 兴奋（Excitement）

**心理学定义：** 兴奋是一种高唤醒度的正向情绪，混合了喜悦和惊讶的期待成分。

**成分构成：**
- **喜悦（Joy）**：50-60%——正向效价基底
- **惊讶（Surprise）**：20-30%——期待性唤醒（anticipatory arousal），对即将发生的事情的新异性
- **恐惧（Fear）**：5-15%——在"惊险兴奋"（thrill/excitement）中出现（如过山车）

**文献依据：** Watson, Clark & Tellegen (1988)的PANAS研究；Clore & Schnall (2005)的情绪认知理论；Frijda (1986)的情绪行动倾向理论。

| **怀旧 (Nostalgia)** | 悲伤（0.35）+ 喜悦（0.35）+ 恐惧（0.15）+ 愤怒（0.15） | — | Sedikides et al. (2008) |
| **讽刺 (Sarcasm/Irony)** | 喜悦（0.30）+ 厌恶（0.30）+ 愤怒（0.25）+ 惊讶（0.15） | — | Gibbs & Colston (2007) |
| **道德愤怒 (Moral Anger)** | 愤怒（0.50）+ 厌恶（0.30）+ 恐惧（0.20） | — | Haidt (2003) |

| **困惑 (Confusion)** | 惊讶（0.40）+ 恐惧（0.30）+ 悲伤（0.20）+ 厌恶（0.10） | — | D'Mello & Graesser (2012) |
| **投入 (Flow)** | 喜悦（0.40）+ 惊讶（0.30）+ 恐惧（0.20）+ 愤怒（0.10） | — | Csikszentmihalyi (1990) |

### 5.3 情绪配方计算框架

基于上述分析，可构建如下量化计算框架：

**公式1：复合情绪评分**
```
E_composite = Σ(w_i × E_basic_i) + Interaction_Terms + Context_Factor
```
其中：
- `w_i` = 各基本情绪的权重（0 ≤ w_i ≤ 1，Σw_i ≈ 1）
- `Interaction_Terms` = 情绪间的交互效应（如恐惧↑时愤怒↑）
- `Context_Factor` = 情境调节因素

**公式2：基于Russell维度的情绪重建**
```
V_composite = Σ(w_i × V_i)
A_composite = Σ(w_i × A_i)
```
- `V_i` = 各成分的基本情绪在效价维度上的值（-1到+1）
- `A_i` = 各成分的基本情绪在唤醒度维度上的值（-1到+1）

**公式3：带效价-唤醒度的基础权重**
```
w_i = α_i × β(Context) × γ(个体差异)
```
- `α_i` = 该情绪的理论基础权重
- `β(Context)` = 情境调节系数
- `γ(个体差异)` = 人格/个体差异调节系数

## 6. 情绪量化方法

### 6.1 主要情绪量表

#### 6.1.1 PANAS（积极-消极情感量表）

**Watson, Clark & Tellegen (1988)**

PANAS是使用最广泛的情绪测量工具，包含两个20题分量表：

**正向情感分量表（Positive Affect, PA）：**
- 活跃的、热心的、兴奋的
- 兴高采烈的、自豪的、欣喜的
- 感兴趣的、精力充沛的、专注的
- 警觉的、灵感迸发的、热情的

**负向情感分量表（Negative Affect, NA）：**
- 痛苦的、愤怒的、恐惧的
- 惊恐的、敌意的、紧张的
- 害怕的、恐惧的、坐立不安的
- 恼怒的、沮丧的、害怕的

**特点：** 效价维度清晰，PANAS中的NA分量表与状态焦虑高度相关（r = 0.85），PA分量表与主观幸福感相关（r = 0.47）

#### 6.1.2 SEARS（状态-特质情绪表达量表）

**State-Trait Emotion Expression Scale (SEARS)**

由Rubin, Feldman & Bloom (1999)开发，测量：
- **状态情绪表达（State Expression）**：即时情绪表达
- **特质情绪表达（Trait Expression）**：习惯性情绪表达风格

#### 6.1.3 DECS（差分情绪量表）

**Differential Emotions Scale (DES)**

由Izard et al. (1983)开发，基于Izard的差分情绪理论（DET）：
- 10种负向情绪分量表
- 4种正向情绪分量表
- 3种中性/复杂性情绪分量表

**信效度：** Cronbach's α = 0.71-0.91（各分量表），重测信度 r = 0.59-0.83

#### 6.1.4 其他重要量表

| 量表名称 | 作者 | 维度/因子 | 题项数 |
|---------|------|---------|-------|
| PANAS | Watson et al. (1988) | 效价（正/负） | 20 |
| DES-IV | Izard et al. (1983) | 10因子 | 40 |
| GSEQ (GEMS) | Gross & John (2003) | 10因子（情绪表达调节） | 16 |
| SEARS | Rubin et al. (1999) | 状态/特质 | 35 |
| EMBU | Perris et al. (1980) | 父母教养知觉（情绪相关） | 81 |
| EQ-i | Bar-On (1997) | 情绪智力15因子 | 133 |
| DERS | Gratz & Roemer (2004) | 情绪调节困难6因子 | 36 |
| GEQ | Bosmans et al. | 情绪导航问卷 | — |

### 6.2 基本情绪评分标准

#### 6.2.1 面部动作编码系统（FACS）

**Ekman & Friesen (1978)**

FACS是客观化基本情绪评分的基础：
- 每种情绪对应特定的面部动作单元（AU）组合
- 强度评级：A（轻微）到E（极致）等级

**基本情绪的AU组合：**

| 情绪 | AU组合 | 描述 |
|------|--------|------|
| 喜悦 | AU6 + AU12 | 脸颊提升 + 嘴角提升 |
| 悲伤 | AU1 + AU4 + AU15 | 眉头上扬 + 眉心收缩 + 嘴角下垂 |
| 愤怒 | AU4 + AU5 + AU7 + AU23 | 眉心收缩 + 上眼睑提升 + 眉头收缩 + 嘴角收紧 |
| 恐惧 | AU1 + AU2 + AU4 + AU5 + AU7 + AU20 | 眉毛上扬 + 上眼睑大幅提升 + 嘴张 |
| 厌恶 | AU9 + AU10 + AU16 | 皱鼻 + 上唇提升 + 下唇下垂 |
| 惊讶 | AU1 + AU2 + AU5 + AU26 | 眉毛上扬 + 眼睛睁大 + 下颌下垂 |

#### 6.2.2 自陈式评分范式

通常采用Likert 5点或7点评分：

```
1 — 完全不感受到该情绪
2 — 轻微感受到该情绪
3 — 中等程度感受到该情绪
4 — 较强感受到该情绪
5 — 强烈感受到该情绪
```

### 6.3 复合情绪的计算方式

#### 6.3.1 维度加权法

基于复合情绪的成分构成，可以采用如下加权计算：

```python
# 伪代码示例：害羞分数计算
def calculate_shyness(fear, joy, disgust, surprise):
    """
    害羞 = 恐惧 + 喜悦 + 厌恶 + 惊讶
    权重: fear=0.40, joy=0.20, disgust=0.30, surprise=0.10
    """
    shyness_score = (
        0.40 * fear +
        0.20 * joy +
        0.30 * disgust +
        0.10 * surprise
    )
    return shyness_score
```

#### 6.3.3 模糊逻辑方法

由于情绪边界具有模糊性，可采用模糊逻辑（Fuzzy Logic）方法：
- 每种基本情绪不是0/1的二元值，而是0-1之间的连续隶属度
- 复合情绪的隶属度由各成分隶属度的隶属函数决定

在情绪计算领域，近年来的进展：

- **基于生理信号的识别**：皮肤电导（GSR）、心率变异性（HRV）、脑电图（EEG）在情绪识别中的应用

## 7. 基本情绪特征描述

### 7.1 喜悦（Joy）

| 维度 | 描述 |
|------|------|
| **身体反应** | 微笑（自发性杜氏微笑/Duchenne smile）、心跳减慢或稳定、皮肤电导降低、肌肉放松、姿态舒展 |
| **心理体验** | 愉悦感、满足感、成就感、对当前情境的积极评价、对未来的乐观期待 |
| **行为倾向** | 趋近行为、社会联结增强、亲社会行为增加、探索行为增加、创造性思维增强 |
| **认知特征** | 注意范围扩大、正向记忆提取优势、对模糊刺激的正向解读、自我效能感增强 |

### 7.2 愤怒（Anger）

| 维度 | 描述 |
|------|------|
| **身体反应** | 心率加快、血压升高、皮肤电导增强、肌肉张力增加（尤其是手部和面部）、呼吸加快、面部血流增加（发红） |
| **心理体验** | 强烈的不快感、对他人的敌意感知、对公正的强烈需求感、主观控制感增加（"准备好行动"） |
| **行为倾向** | 攻击行为（身体/语言）、趋近-攻击倾向、阻止-攻击倾向、冷漠行为（压抑愤怒时）、问题解决行为（工具性愤怒） |
| **认知特征** | 归因偏误（敌意归因偏误）、注意范围收窄至威胁刺激、认知加工加速、正当化思维增强 |

### 7.3 悲伤（Sadness）

| 维度 | 描述 |
|------|------|
| **身体反应** | 心率减慢、呼吸变浅、呼吸频率降低、肌肉张力下降（整个人感觉无力）、声音低沉、眼泪分泌 |
| **心理体验** | 深层的痛苦和失落感、对丧失的反复思考、孤独感、无价值感或内疚感、认知活动减慢 |
| **行为倾向** | 退缩行为（社会退缩）、失去活动动机、沉思倾向（rumination）、寻求社会支持、哭泣 |
| **认知特征** | 负向记忆偏向、注意力偏向负向刺激、时间感知减慢、对自我和未来的负向预期、去个性化思维 |

### 7.4 恐惧（Fear）

| 维度 | 描述 |
|------|------|
| **身体反应** | 心率急剧加快、血压升高、皮肤电导显著增强（冷汗）、肌肉颤抖、呼吸急促、肠道运动加速（应激反应）、瞳孔放大 |
| **心理体验** | 强烈的不安全感、危机感、渺小感、失控感、急切的逃避欲望 |
| **行为倾向** | 防御/回避行为（首要行为倾向）、僵住反应（freeze）、战斗/逃跑/僵住三联反应、解离反应（极端时） |
| **认知特征** | 注意偏向威胁刺激（注意捕获效应）、威胁评估加速、认知收窄（ tunnel vision）、记忆巩固增强（对威胁事件） |

### 7.5 厌恶（Disgust）

| 维度 | 描述 |
|------|------|
| **身体反应** | 面部厌恶表情（皱鼻、上唇提升）、心率可能降低或轻微加快、呼吸变浅、胃肠反应（恶心）、皮肤电导轻度升高 |
| **心理体验** | 对某物的强烈排斥感、"肮脏"或"不洁"的主观感受、需要拉开距离的冲动、对道德违反的强烈反感（道德厌恶） |
| **行为倾向** | 回避行为、驱逐行为（推开）、呕吐反射、身体回避距离增加 |
| **认知特征** | 污染感知增强（contamination）、本质主义思维（感知某物的"本质"被污染）、道德判断强化 |

### 7.6 惊讶（Surprise）

| 维度 | 描述 |
|------|------|
| **身体反应** | 心率短暂加快、瞬时呼吸暂停（O型嘴）、眼睛睁大、眉毛弯曲、觉醒水平瞬间提升、皮肤电导轻度升高 |
| **心理体验** | 对意外事件的即时觉知、新异性感知、认知资源瞬间重新分配、短暂的信息处理中断 |
| **行为倾向** | 行为暂停（停顿反应）、注意指向新异刺激、评估行为（随后的趋近或回避）、预期更新 |
| **认知特征** | 预期-实际结果不匹配检测、工作记忆负荷增加、情境模型更新、注意力重新定向 |

## 8. 总结与计算框架

### 8.1 核心发现

1. **三大理论框架互补：** Ekman的基本情绪理论提供了离散的"情绪原子"，Plutchik的轮模型提供了情绪之间的关系和混合规则，Russell的环状模型提供了连续的维度空间。这三者相互补充，共同构成情绪量化的理论基础。

2. **复合情绪的组成成分存在一定的共识：** 虽然复合情绪的精确构成仍有争议，但各主要情绪模型的交叉验证提供了相对稳定的成分结构。内疚、羞耻、嫉妒等"自我意识情绪"和"道德情绪"的结构已被较充分地研究。

3. **效价-唤醒度模型是量化的基础：** 在缺乏精确成分权重的情况下，基于效价（Valence）和唤醒度（Arousal）的两维度或多维度评分是最可靠的量化方法。

4. **羞耻 = 厌恶（自指向）+ 悲伤 + 恐惧 + 愤怒**的公式得到Lewis (1971)和Tangney (2002)研究的有力支持；**焦虑 = 恐惧 + 悲伤 + 惊讶（预期性）**得到Barlow (2000)研究的支持。

### 8.2 情绪量化建议算法

**基础算法：**
```
输入: 各基本情绪评分 E_i ∈ [0, 1]
复合情绪 C 的评分:

C_score = Σ_i(w_i × E_i)

其中 w_i 来自该情绪的成分权重向量（归一化）
```

**维度约束：**
```
C_V = Σ_i(w_i × V_i)   # 复合效价
C_A = Σ_i(w_i × A_i)   # 复合唤醒度
C_D = Σ_i(w_i × D_i)   # 复合支配度

V_i: [-1, +1]  (负效价到正效价)
A_i: [-1, +1]  (低唤醒到高唤醒)
D_i: [-1, +1]  (被控制到有控制感)
```
